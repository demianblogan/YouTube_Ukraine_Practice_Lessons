    уберіть його з вікна створення проектів Visual Studio

    "My Second Solution" і проект "My First Project".

    поставте відповідну галочку. Ознайомтеся в провіднику з вмістом такого проекту.
    Порівняйте його з проектом із задачі 5, щоб зрозуміти різницю між ними.

9.  Найдіть в любому проекті приховану папку ".vs" і видаліть її. Не забувайте, що для
    цього треба спочатку закрити Visual Studio.

10. Створіть файл "MyFirstFile.cpp" в папці Source Files у любому вашому проекті.

11. Створіть в цьому ж проекті ще один файл, тільки у папці Resource Files. Після цього
    перемістіть його в Source Files.

12. Налаштуйте розмір шрифта коду під себе, використовуючи масштабування.

13. Знайдіть в провіднику ваші .cpp файли.

14. Запустіть любе створене вами рішення не через Visual Studio, а через ярлик в провіднику.

15. Запустіть любий створений вами проект не через Visual Studio, а через ярлик в провіднику.

16. Перейменуйте файл із задачі 9, давши йому ім'я "main.cpp".

17. Перейменуйте любий проект у Visual Studio. Перевірте, чи в провіднику змінилося його ім'я.

18. Видаліть файл із задачі 10 із проекту у Visual Studio.

19. Уберіть любий проект із Visual Studio, після чого видаліть його в провіднику.

20. Уберіть любий файл із Visual Studio, а потім добавте його назад у проект.

21. Уберіть любий проект із Visual Studio, а потім добавте його назад у рішення.

22. Напишіть у вашому .cpp файлі код int main(){}. Скомпілюйте цей проект, а потім запустіть його.
