1. Запустіть любий проект через дебагер, щоб перевірити, чи у вас він працює. Спочатку
   запустіть дебагер через натискання пункту меню Debug -> Start Denugging. Якщо все спрацювало,
   закрийте дебагер через натискання червоного квадрату.

2. Аналогічна задача, тільки запустіть дебагер через клавішу F5, після чого закрийте через
   Shift + F5.

3. Напишіть наступний код:

	#include <iostream>
	int main()
	{
		std::cout << "Enter your age: ";
		int age;
		std::cin >> age;
		std::cout << "Your age is " << age << std::endl;
		return 0;
	}

   Покрокуйте по цьому коду рядок за рядком завдяки дебагеру. Подивіться, як міняється значення
   змінної age до введення значення і після.

4. В попередній програмі поставте точку-останову на рядку 
   "std::cout << "Your age is " << age << std::endl;". Запустіть дебагер. У вас має працювати
   програма так, як завжди, але після введення значення програма зупиниться на точці-остановій.

5. Напишіть наступний код:

	#include <iostream>
	int main()
	{
		std::cout << "How many iterations do you want?: ";
		int iterations;
		std::cin >> iterations;
		for (int i = 0; i < iterations; i++)
		{
			std::cout << "Iteration #: " << i << std::endl;
		}
		return 0;
	}

	Завдяки клавіші F10 запустіть дебагер і пройдіться рядок за рядком по всьому коду.
