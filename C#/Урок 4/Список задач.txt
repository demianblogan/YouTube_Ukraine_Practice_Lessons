1. Створіть C# Console проект з іменем "My First Created Project"

2. Перейменуйте його в Visual Studio на "Project 1"

3. Разом з проектом створилося рішення. Додайте ще один проект в це рішення під іменем "Project 2".
   УВАГА: якщо проект не можна додати, значить ви при створенні першого проекту указали, щоб рішення
   і проект розміщалися в одній папці. В такому разі зніміть цю галочку і заново виконайте пункт 1.

4. Видаліть проект "Project 1" із рішення. Не забудьте також видалити його в провіднику.

5. Запустіть проект "Project 2"

6. Перейменуйте файл "Program.cs" в проекті "Project 2" на любе інше ім'я, наприклад "MyFile.cs"
   (не забудьте залишити розширення файлу). Познайомтеся ближче з вікном підтвердження і подивіться,
   що міняється в коді, якщо натиснути кнопку "Так" в цьому вікні.

7. Закрийте Visual Studio і видаліть рішення із провідника. Запустіть вікно вибору проектів в Visual Studio
   (перше вікно, яке появляєтьтся при запуску Visual Studio) і спробуйте відкрити проект, який показаний зліва.
   Проаналізуйте повідомлення, яке появиться.

8. В списку шаблонів проектів найдіть шаблон "Blank Solution". Це просто пусте рішення без жодний проектів.
   Створіть цей "проект".

9. Добавте до пустого рішення 2 проекти. Назвіть їх так, як хочете.

10. В першому проекті в файлі Program.cs напишіть наступний код:

	class Program
	{
		static void Main()
		{
			Console.BackgroundColor = ConsoleColor.Blue;
			Console.ForegroundColor = ConsoleColor.Yellow;
			Console.WriteLine("Hello from project 1!");
		}
	}

11. В другому проекті напишіть ось цей код:

	class Program
	{
		static void Main()
		{
			Console.BackgroundColor = ConsoleColor.Black;
			Console.ForegroundColor = ConsoleColor.White;
			Console.WriteLine("Hello from project 2!");
		}
	}

12. Запустіть спочатку перший проект, а потім другий. У вас мають відрізнятися колір фону і тексту
    в консольних вікнах.

13. Видаліть в першому проекті файл "Program.cs". В цьому ж проекті створіть новий файл з таким же
    кодом, який був в пункті 10 і запустіть проект. Якщо все правильно зробите - проект має запуститися.

14. Налаштуйте розмір шрифту та налаштування консолі під себе.
